# TING

Este repositório contém o projeto "_TING_", requisito em atendimento ao módulo de Ciẽncia da Computação, da Trybe. O projeto consiste em implementar um programa que simule um algoritmo de indexação de documentos similar ao do Google.

### Habilidades:
 - Manipular Pilhas;
 - Manipular Deque;
 - Manipular Nó & Listas Ligadas e;
 - Manipular Listas Duplamentes Ligadas.
